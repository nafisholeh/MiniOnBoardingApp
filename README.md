# Onboarding App

This app built for Front-End (UI-based) Code Challenge as part of interview process.

1. Download "Expo" app on Google Play or App Store
2. Open "Expo" app
3. Click "Scan QR Code" button
4. Scan the Barcode and you are ready to go:
